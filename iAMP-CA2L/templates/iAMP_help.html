iAMP-CA2L Server User Guide

Descript

iAMP-CA2L is a two-layer classifier. The first layer is used to determine whether an unknown peptide is an antimicrobial peptide, and the second layer is used to determine the specific function of an antimicrobial peptide.

Intput sequence

After the user submits a sequence, click the Submit Sequence button, and the final prediction result will be directly displayed on the web page.

Content description

(Antibacterial,Antiviral,Antiparasital,Anti-HIV,Anticancer(antitumor),anti-MRSA,Antifungal,anti-endotoxin,Antibiofilm,Chemotactic)
